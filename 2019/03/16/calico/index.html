  1. 将机器上的ip转发功能打开

5）修改docker启动文件以支持etcd

node1

在ExecStart区域内添加 (在–seccomp-profile 这一行的下面一行添加–cluster-store=etcd://192.168.15.133:2379 \ )

node2

在ExecStart区域内添加 (在–seccomp-profile 这一行的下面一行添加–cluster-store=etcd://192.168.15.134:2379 \ )
